## How to setup instances

### application node (m3.large)

- install java7 (devel)
- download node
- add node to PATH
- download maven
- add maven to PATH
- "too many open files"
- checkout project
- run

- install java7 (devel)
- download gatling
- fix "too many open files"
- checkout project
- delete scala file
- run
